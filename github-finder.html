Github Finder


The Github Finder is a React app that searches for Github profiles and their profile details.

The app searches for the keyword in the username and the bio and once the profile is selected the app navigates to a new page and shows most of the users details like, location, latest public repositories, hireable status.

The app connects to the Github API to fetch all the user data.

Technologies:
  • - React
  • - HTML
  • - CSS
