ADOPT ME - A pet adoption app.


Adopt Me is a React app that searches for adoptable pets.

The App fetches data from a public API to prompt the results and contact of the pet owners.

The App was created along with the frontend masters course "Complete Intro to React v5" by Brian Holt. I chose to participate in this course to practice the React eco system. Including hooks, effects, context, Reach router and portals.

Technologies:
  • - React
  • - HTML
  • - CSS
