Sound of Silence

The Sound of Silence is a vanilla javascript search app for songs with the title "Sound of Silence".
Each entry shows: the track name, artist name, artwork, release date price, and currency.
The artist data is stored in an internal database in the file music.js.

Technologies:
  • - JavaScript
  • - HTML
  • - CSS
  • - bootstrap
